## Работа с проектом

---

### Запуск проекта в докере

Для корректной работы почты добавить свои настройки в .env

Запуск контейнеров проекта  
`docker compose up --build`

Остановка контейнеров  
`docker compose down -v`

Проект доступен по адресу:  
[http://localhost](http://localhost)

GraphQL доступен по адресу:  
[http://127.0.0.1:8000/graphql](http://127.0.0.1:8000/graphql)

### Запуск проекта локально
Предварительно надо добавить в файл .env данные с настойками почты
Далее необходимо виртуальное окружение для работы с проектом

`python -m venv .venv`

`pip install -m requirements.txt`

`python manage.py runserver`

Проект доступен по адресу:  
[http://127.0.0.1:8000](http://127.0.0.1:8000)

## Структура проекта
```text
project/
├── accounts              # Приложение для работы с профилем пользователя
├── logs                  # Папка с логами проекта
├── management            # Приложение кастомной админки проекта
├── media                 # Динамические файлы проекта
├── nginx                 # Конфигурация прокси сервера
├── orders                # Приложение заказов и работы с корзиной
├── products              # Приложение с категориями и товарами магазина
├── shop                  # Главная папка проекта с настройками
├── static                # Папка проекта с собранной статикой
├── static_src            # Папка со статикой хранящая исходники
├── templates             # Папка с базовыми шаблонами дизайна
├── requirements.txt      # Зависимости Python
├── .env                  # Данные настроек контейнера
├── Dockerfile            # Dockerfile для сборки образа
├── docker-compose.yml    # Конфигурация Docker Compose
├── db_fixtures.json      # База для приложения (и для тестов)
├── manage.py             # Точка входа и менеджер Django
├── pytest.ini            # Настройки для тестов
├── README.md             # Описание проекта
├── Task.md               # Техническое задание
```

## Маршруты

```
/                                  # Главная страница (она же каталог)
/accounts/login/                   # Страница формы авторизации
/accounts/logout/                  # Адрес выхода из системы
/accounts/register/                # Страница формы регистрации
/accounts/orders/                  # Страница списка заказов пользователя
/accounts/profile/                 # Форма редактирования профиля пользователя
/accounts/security/                # Форма смены пароля
/accounts/balance/                 # Форма пополнения баланса профиля
/accounts/password-reset/          # Форма сброса пароля
/accounts/password-reset/done/     # Сообщение об успешной отправке ссылки восстановления
/accounts/reset/<uidb64>/<token>/  # Одноразовый адрес для восстановления пароля
/accounts/reset/done/              # Сообщение об успешном восстановлении пароля

/admin/                            # Dashboard кастомной админки
/admin/products/                   # Список товаров для администрирования
/admin/product/add/                # Добавление нового товара
/admin/product/<slug:slug>/        # Форма редактирования товара
/admin/categories/                 # Список категорий для администрирования
/admin/category/add/               # Добавление новой категории
/admin/category/<slug:slug>/       # Форма редактирования категории
/admin/category/delete/<int:pk>/   # Удаление категории по ID
/admin/users/                      # Список пользователей для администрирования
/admin/reviews/                    # Список отзывов для администрирования

/orders/cart/                      # Страница корзины товаров
/orders/checkout/                  # Страница заказа товаров

/products/                         # Страница каталога товаров
/products/guides/                  # Страница какая то...
/products/add-review/              # Страница добавления отзывов
/products/<slug:slug>/             # Страница детальной информации товара

/administrator/                    # Стандартная админка сайта

/graphql/                          # Документация и описание GraphQL запросов
```
